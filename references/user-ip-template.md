# 用户 IP 信息模板

建议至少提供下面这些信息，越具体，选题和文案越像你本人：

## 1. 账号定位
- 你的小红书账号想做什么方向？
- 你希望别人一想到你，会想到什么关键词？

## 2. 目标受众
- 你想吸引谁？
- 他们现在最痛的点是什么？

## 3. 个人经历 / 背景
- 你的职业、经历、成绩、踩坑、转型故事
- 哪些是真实发生在你身上的？

## 4. 擅长领域
- 你最能讲明白什么？
- 你有什么别人没有的一手经验？

## 5. 商业目标
- 你是为了涨粉、接单、卖课、卖产品，还是做个人品牌？

## 6. 语气风格
- 你偏理性、犀利、真诚、专业、松弛，还是像朋友聊天？

## 7. 禁区 / 边界
- 哪些内容你不想碰？
- 哪些表达方式你不喜欢？
